# Персональний помічник 
## Віртуальне середовище

Для створення віртуального середовища було використано pipenv.

Команда створення середовища:
pipenv --python 3.14

У файлі Pipfile зафіксована версія Python:

[requires]
python_version = "3.14"

Сторонні бібліотеки у проекті не використовуються, тому розділ [packages] порожній.

IDE налаштовано для роботи з pipenv interpreter.
